# Aion AI Agency Landing Page v2 - Enterprise Lead Quality

Production-grade landing page for Fortune 500 AI deployment services.

## Overview
- **Target:** COOs, VP Operations, Chief Digital Officers at B+ companies
- **Core Message:** Deploy AI systems that cut operational costs by 20-40% in 90 days
- **Design:** Premium restraint, near-black + white, enterprise-focused

## Tech Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- React Hook Form + Zod
- Calendly integration

## Deployment
- Platform: Vercel
- Performance target: <2s load time
- Mobile-first, responsive design
